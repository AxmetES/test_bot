[Ведущему: разборчиво прочитать слово "септИнов".]
   Исследование показало, что в приложениях ко многим статьям по
генетике содержатся ошибки, которые коснулись, например, септИнов и
идентификатора 201E9 [два, ноль, один, е, девять]. Эти ошибки вызваны
тем, что многие авторы и рецензенты статей недостаточно хорошо
знакомы... С чем?

Ответ:
С интерфейсом Microsoft Excel [чтецу: майкрософт эксель].

Зачет:
По упоминанию Microsoft Excel [чтецу: майкрософт эксель], LibreOffice
Calc [чтецу: либра офис кальк], OpenOffice Calc [чтецу: оупен офис
кальк] или электронных таблиц.

Комментарий:
Приложения к статьям по генетике - таблицы с данными, которые
составляются в Microsoft Excel [майкрософт эксель] и других аналогичных
программах. В них по умолчанию используется общий формат ячеек, который
автоматически конвертирует многие введенные значения. Например,
антибиотики септины, которые обозначаются "SEPT1" [септ, один], "SEPT2"
[септ, два], Excel [эксель] считает обозначениями дней сентября. А
идентификаторы вроде 201E9 [два, ноль, один, е, девять] Excel [эксель]
конвертирует в числа с плавающей запятой.

Источник:
   1. Ziemann M., Eren Y., El-Osta A. Gene name errors are widespread in
the scientific literature // Genome Biology. - 2016. - Т. 17. - N 1. -
С. 177.
https://genomebiology.biomedcentral.com/articles/10.1186/s13059-016-1044-7
   2. https://geektimes.ru/post/279766/
